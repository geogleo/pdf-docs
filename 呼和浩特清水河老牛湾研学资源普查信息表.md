## 二、地理位置与交通

### 2.1 地理位置特征

| 特征项 | 描述 |
|--------|------|
| 地理位置 | 位于山西省偏关县与内蒙古清水河县交界处 |
| 地貌特征 | 黄河大峡谷、黄土高原沟壑地貌 |
| 水系特征 | 黄河由北向南流经，形成"S"形大弯 |
| 边界特征 | 内蒙古与山西两省区交界，"鸡鸣闻两省" |
| 周边景点 | 老牛湾古堡、乾坤湾、明长城遗址 |

### 2.2 交通信息

| 交通方式 | 路线说明 | 时间/距离 |
|----------|----------|-----------|
| 自驾（呼和浩特出发） | 呼和浩特 → 呼准高速 → 清水河县 → 老牛湾 | 约3小时，150公里 |
| 自驾（太原出发） | 太原 → 大运高速 → 忻州 → 偏关 → 老牛湾 | 约3.5小时，200公里 |
| 公共交通 | 呼和浩特客运站乘班车至清水河县，转乘当地车辆 | 约4小时 |
| 飞机 | 呼和浩特白塔国际机场，转乘汽车 | 约4小时 |
| 火车 | 呼和浩特东站 → 清水河县站 → 转乘汽车 | 约3.5小时 |

## 五、研学课程设计建议

### 5.1 主题课程

| 课程名称 | 适合学段 | 课程目标 | 主要内容 |
|----------|----------|----------|----------|
| 黄河的奥秘 | 小学3-6年级 | 了解黄河地理特征 | 黄河起源、流经区域、地貌特征 |
| 黄土高原探秘 | 初中 | 认识黄土高原成因与生态 | 黄土成因、水土流失、生态保护 |
| 长城与黄河 | 初中/高中 | 理解军事防御与地理关系 | 明长城、军事防御、地理优势 |
| 地质考察 | 高中 | 掌握野外地质调查方法 | 地层观察、地貌测量、地质素描 |
| 黄河文化寻根 | 初中/高中 | 理解黄河在中华文明中的地位 | 黄河流域文明、农耕文化 |
| 古堡历史探秘 | 初中 | 了解明代边塞军事文化 | 古堡建筑、军事防御、边塞生活 |

### 5.2 实践活动设计

| 活动名称 | 活动类型 | 时长 | 活动内容 |
|----------|----------|------|----------|
| 黄河水质监测 | 科学实践 | 2小时 | 采集水样、检测pH值、观察水质 |
| 黄土地貌素描 | 美术实践 | 2小时 | 观察地貌、绘制素描、记录特征 |
| 古堡建筑测量 | 数学实践 | 2小时 | 测量建筑尺寸、绘制平面图 |
| 民歌采风 | 文化实践 | 3小时 | 采访当地老人、学唱民歌、记录歌词 |
| 徒步探险 | 体育实践 | 4小时 | 沿黄河徒步、观察地貌、团队协作 |
| 星空观察 | 科学实践 | 2小时（夜间） | 观察星空、认识星座、学习天文知识 |

## 七、安全保障

### 7.1 安全风险

| 风险类型 | 具体描述 | 防范措施 |
|----------|----------|----------|
| 地质风险 | 黄土高原沟壑，可能发生滑坡 | 避开危险区域，设置警示标志 |
| 水域风险 | 黄河水深流急，危险水域 | 设置警戒线，配备救生设备 |
| 天气风险 | 夏季高温、雷雨，冬季寒冷 | 关注天气预报，合理安排行程 |
| 交通风险 | 山路崎岖，部分路段较险 | 使用正规车辆，减速慢行 |

### 7.2 安全保障措施

| 措施类型 | 具体内容 |
