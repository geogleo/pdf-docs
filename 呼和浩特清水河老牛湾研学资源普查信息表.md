# 呼和浩特清水河老牛湾研学资源普查信息表

（自然生态类、历史文化类，150124ZR001）

| 普查项目 | 具体信息内容 |
|----------|--------------|
| 一、点位官方名称 | 老牛湾黄河大峡谷旅游区（老牛湾国家地质公园） |
| 二、所在行政区划 | 内蒙古自治区呼和浩特市清水河县（单台子乡） |
| 三、经纬度坐标 | 东经：111°26′，北纬：39°55′（核心区参考坐标） |
| 四、基地/营地属性 | 国家4A级旅游景区、国家地质公园（属于"自然地质+黄河文化+长城文化"复合型资源，拥有"黄河与长城握手"的独特地标） |
| 五、接待人数能力 | 日最大接待量：5,000人次  
研学团队承载力：核心观景台及古村落区域空间有限，建议单次接待 300-500 人，需分批次错峰进入核心区。 |
| 六、交通便捷可达性 | 评价：一般（偏远）  
1.公路：距离呼和浩特市区约 170 公里，车程约 2.5-3 小时，路况多为盘山公路。  
2.停车：拥有大型生态停车场，可容纳大巴车。  
3.限制：距离市区较远，往返时间长，不适合半日游或短途研学，通常需安排两日行程。 |
| 七、餐饮接待能力 | 评价：优（地域特色浓）  
1. 特色餐饮：周边农家乐发达，主打黄河鲤鱼、炸油糕、酸米饭等晋陕蒙特色美食。  
2.接待规模：景区内及入口处有多家大型农家院，具备同时接待 200-300 人团餐的能力。 |
| 八、住宿接待能力 | 评价：良好（特色住宿）  
1.特色体验：拥有窑洞宾馆、特色民宿及星空帐篷营地，能够提供特色的"窑洞住宿"体验。  
2.规模：床位总数约 500-800 张，能满足中小型研学团队的过夜需求。 |
| 九、研学教室情况 | 评价：户外为主，室内为辅  
1.室内场馆：地质公园博物馆，可开展地质科普讲座。  
2.户外课堂：黄河峡谷观景台、古村落遗址、明长城遗址、游船码头，是天然的历史与地理大课堂。 |
| 十、科普设施情况 | 评价：卓越  
1.地质奇观：典型的黄土高原地貌、峡谷地貌、流水侵蚀地貌，是地理学的天然教科书。  
2.历史遗迹：保存完好的明代敌楼（望河楼）、古村落石板路。  
3.设施：游船、索道（部分区域）、玻璃栈道等体验设施。 |
| 十一、安全保障条件 | 评价：中高（需重点防范）  
1.地形风险：景区多为悬崖峭壁、峡谷水边，需严格的封闭式管理和老师陪同，防跌落、防溺水是重中之重。  
2.救援：偏远山区，医疗救援响应时间较长，需配备随队医护人员。 |
| 十二、研学师资条件 | 评价：专业  
1.专职讲解：景区导游熟悉地质成因与历史典故。  
2.学科导师：极适合地理、历史学科教师实地授课，是跨学科（STEAM）研学的理想场所。 |
| 十三、课程开发情况 | 已开发成熟课程主题：  
1.地质地理类：《黄河峡谷成因探秘》、《黄土高原地貌观察》、《岩石的奥秘》。  
2.历史文化类：《长城与黄河的握手》、《明代边防体系调研》、《古村落建筑美学（窑洞）》。  
3.非遗民俗类：《捏面人/剪纸体验》、《黄河谣传唱》。 |
| 十四、适合学段 | 侧重初高中（地理野外考察、历史实地探究）。不建议低龄段参加（因路途遥远及地形风险）。 |
| 十五、综合评价 | 核心优势：  
1.资源唯一性："长城与黄河交汇"的绝版景观，具有极高的地理与历史研学价值。  
2.体验感强：乘船游览黄河、住窑洞、登长城，体验维度丰富。  
存在短板：  
1.交通瓶颈：距离呼和浩特市区远，往返耗时过长，对车辆和司机要求高。  
2.季节限制：冬季黄河封冻，部分项目暂停，旺季集中在春夏秋三季。  
3.设施分散：景点之间距离较远，需车辆接驳。  
普查结论：属于稀缺级研学资源，是呼和浩特市研学版图中"黄河文化"板块的核心支撑，适合开展深度二日研学营地活动。  
发展建议：深挖地质文化课程，完善食宿配套，打造两日营地。 |
